# doppio_pendolo
Questo codice simula un pendolo doppio in Python. Utilizza la funzione odeint di SciPy per risolvere le complesse equazioni del moto del pendolo doppio e Matplotlib per tracciare e animare i risultati. Puoi regolare i parametri come la gravit√†, la lunghezza e la massa dei pendoli per vedere come influenzano il movimento.
